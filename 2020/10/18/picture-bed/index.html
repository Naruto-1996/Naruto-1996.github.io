PicGo下载地址

2、创建一个Github仓库并进行设置

（1）创建仓库

(2) 点击头像 点击Settings

(3) 再点 Developer settings

(4) 再点 Personal access tokens

(5) 填写内容

(6) 复制生成的token (注意：网页关闭后就再也看不到这个token了)

3、在刚才创建的仓库中新增一个项目

git init
git add .
git commit -m "备注"
git remote add origin 仓库地址
git push -u origin master
